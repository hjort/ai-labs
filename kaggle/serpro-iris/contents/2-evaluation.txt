# Objetivo

É seu trabalho prever qual das três diferentes espécies pertence cada uma das observações.
Para cada flor no conjunto de testes, você deve prever a variável `Species` com uma dessas opções: 'Iris-setosa', 'Iris-versicolor' ou 'Iris-virginica'.

# Métrica

Sua pontuação é a porcentagem de flores íris que você prevê corretamente. Isso é conhecido simplesmente como precisão.

# Formato do arquivo de submissão

Você deve enviar um arquivo CSV com exatamente 50 registros mais uma linha de cabeçalho (*header*). Seu envio mostrará um erro se você tiver linhas ou colunas extras.

O arquivo deve ter exatamente 2 colunas:

- `Id` (classificado em qualquer ordem)
- `Species` (contém a previsão da espécie: 'Iris-setosa', 'Iris-versicolor' ou 'Iris-virginica')

Você pode fazer o download de um arquivo de envio de exemplo (`iris-example.csv`) na <a href="https://www.kaggle.com/c/serpro-iris/data">página de dados</a>.

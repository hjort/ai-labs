Os dados foram divididos em dois grupos:

- **conjunto de treinamento** (`weather-train.csv`)
- **conjunto de testes** (`weather-test.csv`)

O **conjunto de treinamento** deve ser usado para construir seus modelos de aprendizado de máquina. Para o conjunto de treinamento, fornecemos o resultado (também conhecido como a “verdade fundamental”) para cada observação (neste caso, a temperatura média diária - campo `temperature`). Seu modelo preditivo será baseado nos atributos ou *features* (neste caso, somente a data - especialmente mês e dia).

O **conjunto de testes** deve ser usado para medir o desempenho do seu modelo em dados desconhecidos. Para o conjunto de testes, não fornecemos a verdade básica para cada observação (neste caso, o valor exato da temperatura média). É seu trabalho predizer esses resultados. Para cada observação do teste, use o modelo que você treinou para prever a temperatura média diária.

Também incluímos o arquivo `weather-example.csv`, um **conjunto de previsões fictício** (com resultados aleatórios), como um exemplo de como deve ser um arquivo de envio (submissão) a ser automaticamente avaliado. Esse arquivo possui apenas dois campos: `date` (a data) e `temperature` (o valor da temperatura em graus Celsius).
